# Contact & Support Configuration

## Support Email
- **Address**: support@fairshift.app
- **Purpose**: User support, bug reports, feature requests
- **App Location**: Settings → Contact Support

## In-App Feedback Channels
1. **Settings → Contact Support**: Opens mailto:support@fairshift.app
2. **Settings → Privacy Policy**: Links to https://fairshift.app/privacy
3. **Settings → Terms of Use**: Links to https://fairshift.app/terms

## App Store Connect
- **Review Response**: Monitor and respond to App Store reviews
- **Version Release Notes**: Update with each submission

## Support Workflow
1. User taps "Contact Support" in Settings
2. Mail compose sheet opens with pre-filled recipient
3. User sends email
4. Team responds within 24-48 hours

## Notes
- No in-app feedback form (uses system mail composer for simplicity)
- No crash reporting SDK (rely on App Store Connect crash reports)
- No analytics SDK (privacy-first approach)
